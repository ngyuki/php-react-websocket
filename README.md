PHP React WebSocket

Example
--------------------------------------------------------------------------------

 - [example/index.html](example/index.html)
 - [example/ws.php](example/ws.php)

```console
$ git clone https://github.com/ngyuki/react-websocket.git
$ cd react-websocket
$ php example/ws.php
```

Open `http://localhost:8888/` by your browser.
